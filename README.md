# linebot-todo
LINE bot for helping people manage their todo list

# Installation

```bash
$ npm install
```

Setup below environment variables

```bash
CHANNEL_ID
CHANNEL_SECRET
CHANNEL_ACCESS_TOKEN
WEB_CHANNEL_ID
WEB_CHANNEL_SECRET
WEB_CALLBACK_URL
```

Start the server
```bash
$ npm start
```
